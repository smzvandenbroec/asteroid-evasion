Requirements:
 - A (virtual machine) Linux environment; This game does not work on WSL or other windows subsystems!
 - A lot of free time as you will definitely enjoy this game

----------------------------------------------------------------------------------------------------------------

How to play:
 1. Use the cd/ls commands to go to the current directory.
 2. Compile the game.s file by using the following Linux command:
    'gcc -no-pie -o game game.s'.
 3. Run the compiled file by using the following Linux command:
    './game'.
 4. Use the number keypads to select menu items.
 5. Enjoy!
